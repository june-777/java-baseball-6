- [ ] 힌트 얻기

    - [ ] 같은 수가 다른 자리에 있으면 볼
    - [ ] 같은 수가 전혀 없으면 낫싱

- [ ] 컴퓨터가 선택한 3개의 숫자를 모두 맞히면 게임이 종료

- [ ] 게임 종료 후, 게임 완전히 종료

    - [ ] 숫자 야구 게임을 시작합니다.
    - [ ] 숫자를 입력해주세요 :
    - [ ] 힌트 출력
    - [ ] 3개의 숫자를 모두 맞히셨습니다! 게임 종료
    - [ ] 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
- [ ] 콘솔 입력 기능
    - [ ] 서로 다른 3개의 숫자를 입력한다.
    - [ ] 게임 종료 시, 1 또는 2를 입력한다.
